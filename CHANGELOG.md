# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-16

### Added

- Initial public release
- Real-time heart rate monitoring with live charts
- SpO2 blood oxygen saturation monitoring
- Step tracking with daily accumulation
- Battery level monitoring
- Activity data tracking (15-minute intervals)
- Real-time 3-axis accelerometer visualization
- Web Bluetooth connection management
- Responsive dashboard layout
- Connection status indicators
- Data quality metrics
- Debug information panel
- Comprehensive documentation
- Contributing guidelines
- Code of conduct
- Security policy
- MIT License

### Features

- Direct Bluetooth connection to Colmi R02/R09 rings
- No backend required - fully client-side application
- Modern UI built with Tailwind CSS
- TypeScript type safety throughout
- Next.js 16 App Router
- React 19 with hooks
- Recharts for data visualization
- Lucide React icons

### Documentation

- Complete README with setup instructions
- Contributing guidelines
- Code of conduct
- Security policy
- API documentation in code comments
- Project structure documentation

### Development

- ESLint configuration
- Prettier code formatting
- TypeScript strict mode
- EditorConfig for consistent coding style
- VSCode recommended extensions
- Git ignore rules
- Environment variable template
